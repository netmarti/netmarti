# Auditoría Semanal de Repositorios Destacados

*Esta es una plantilla para auditorías automáticas de calidad y relevancia.*

# Criterios de Calificación

| Factor | Peso | Umbral Mínimo |
|--------|------|---------------|
| Actividad reciente (30d) | 40% | ≥1 commit |
| Documentación completa | 30% | README + CONTRIBUTING |
| Validez temática (IA/Agents) | 20% | Related topic match |
| Estrellas orgánicas | 10% | ≥3 stars |

# Proceso de Decisión

1. Revisión Automática: El bot analiza repositorios nuevos
2. Alerta Semanal: Notificación cada Miércoles
3. Decisión Humana: Tú apruebas/descartes sugerencias
4. Actualización Manual: Vas a Settings → Featured Repositories

# Checklist para Aprovechar al Máximo

- [ ] Revisar reporte cada Miércoles (5 min)
- [ ] Marcar candidatos de reemplazo en GROWTH-TRACKER.md
- [ ] Actualizar Featured Repositories si aplica
- [ ] Limpiar forks inactivos (>90 días sin commits)


_** Este es un sistema de asistencia a la decisión, no automatiza cambios críticos sin supervisión humana. **_

_Mantenimiento repositorios._
